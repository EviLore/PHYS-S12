Final Project Proposal: Digilogue

This document proposes a final project named Digilogue. The project will involve fabricating a wearable musical instrument by embedding electronics in a glove. The glove can then be used to perform commands using an accelerometer and gyroscope for gestural controls, as well as commands using the digits.
A digital synthesizer can be loaded onto an arduino controller to act as the core instrument.
Additionally, it may be feasible to include a sequencer to save a series of notes for playback.


Let's look over a diagram of the proposed device.



It will be constructed with a glove as the base layer. On the back of the hand is an Arduino Uno R3 connected to a small external battery pack for untethered power. A flex sensor is attached to each finger, which will determine finger position and translate that input into output from the synthesizer.
